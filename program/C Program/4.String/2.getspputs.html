C gets() and puts()


The gets() and puts() are declared in the header file stdio.h. Both the functions are involved in the input/output operations of the strings.

* gets() និង puts() ត្រូវបានប្រកាសនៅក្នុង header ថាឯកសារ stdio.h ។ មុខងារទាំងពីរនេះត្រូវបានចូលរួមនៅក្នុងប្រតិបត្តិការបញ្ចូល / បង្ហាញលទ្ធផលនៃតួរអក្ស។

C gets()

The gets() function enables the user to enter some characters followed by the enter key. All the characters entered by the user get stored in a character array. The null character is added to the array to make it a string. The gets() allows the user to enter the space-separated strings. It returns the string entered by the user.

* មុខងារ gets() អាចឱ្យអ្នកប្រើបញ្ចូលតួអក្សរមួយចំនួនបន្តដោយគ្រាប់ចុចបញ្ចូល។ រាល់តួអក្សរដែលបញ្ចូលដោយអ្នកប្រើត្រូវបានរក្សាទុកនៅក្នុងជួរតួអក្សរ។ តួអក្សរទទេត្រូវបានបន្ថែមទៅ array ដើម្បីឱ្យវាក្លាយជាអក្សរ។ gets() អនុញ្ញាតឱ្យអ្នកប្រើបញ្ចូល space-separated strings។ វាត្រឡប់ string ដែលបញ្ចូលដោយអ្នកប្រើប្រាស់។

Declaration

  • char [] gets( char [] );
  • //Declaration gets;

Example

 

+ Reading string using gets()

  • #include  < stdio.h >
  • int main()
  • {
  • char s= [30];
  • printf( ("Enter the string?" );
  • gets(s);
  • printf( "You entered %s" ,s);
  • return 0;
  • }

 

Output1:

  • Enter the string? Code Share is the best
  • Learning Website You entered Code Share is the best

 

The gets() function is risky to use since it doesn't perform any array bound checking and keep reading the characters until the new line (enter) is encountered. It suffers from buffer overflow, which can be avoided by using fgets(). The fgets() makes sure that not more than the maximum limit of characters are read. Consider the following example.

* មុខងារ gets() គឺប្រថុយប្រថានក្នុងការប្រើព្រោះវាមិនអនុវត្តមានការត្រួតពិនិត្យនិងរក្សាអក្សររហូតដល់បន្ទាត់ថ្មី (បានបញ្ចូល) ត្រូវបានជួបប្រទះ។ វាត្រូវបានជៀសវាងដោយកាប្រើ fgets () ។ fgets () ធ្វើកាប្រាកដជាក់លាក់មិនលើសពីចំនួនអតិបរមានៃតួអក្សរត្រូវបានអាន។

+ Reading string using gets() 2

  • #include  < stdio.h >
  • int main()
  • {
  • char str= [50];
  • printf( ("Enter the string?" );
  • fgets(str, 20, stdin);
  • printf( "%s" ,str);
  • return 0;
  • }

 

Output1:

  • Enter the string? Code Share is the best
  • website Learning Code Share is the best

C puts()

The puts() function is very much similar to printf() function. The puts() function is used to print the string on the console which is previously read by using gets() or scanf() function. The puts() function returns an integer value representing the number of characters being printed on the console. Since, it prints an additional newline character with the string, which moves the cursor to the new line on the console, the integer value returned by puts() will always be equal to the number of characters present in the string plus 1.

* មុខងារ puts() គឺស្រដៀងនឹងមុខងារ printf () ។ មុខងារ puts() ត្រូវបានប្រើដើម្បីបោះពុម្ពអក្សរនៅលើ console ដែលត្រូវបានអានពីមុនដោយប្រើមុខងារ gets() ឬ scanf() ។ មុខងារ puts() ផ្ដល់មកវិញនៅ តម្លៃចំនួនគត់តំណាថសឱ្យ ចំនួនតួអក្សរដែលត្រូវបានបង្ហាញនៅលើ console។ ចាប់តាំងពីវាបង្ហាញតួអក្សរថ្មីបន្ថែមជាមួយអក្សរដែលផ្លាស់ថ្មីនៅលើ​ console តម្លៃចំនួនគត់បានត្រឡប់មកវិញដោយ puts() នឹងស្មើនឹងចំនួនតួអក្សរដែលមាននៅក្នុងអក្សរបូក ១ ជានិច្ច។

Declaration

  • int [] puts( char [] );
  • //Declaration gets;

 

Let's see an example to read a string using gets() and print it on the console using puts().

Example:

  • #include  < stdio.h >
  • #inlcude  < string.h >
  • int main()
  • {
  • char name[50];
  • printf( "Enter your name: " );
  • gets(name);
  • printf( "Your name is: " );
  • puts(name);
  • return 0;
  • }

 

Output1:

  • Enter your name: Im Sambathratanak
  • Your name is: Im Sambathratanak
