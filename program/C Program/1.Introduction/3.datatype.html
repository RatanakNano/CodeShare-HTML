DataType in C programming


A data type specifies the type of data that a variable can store such as integer, floating, character, etc.

Type DataType
Type of Data int, char, float,double
Enumeration enum
Data type complex Pointer, array, structure, union
Data void void
DataType Size
char 1 byte
int 2 bytes
float 4 bytes
Double 8 bytes

Integer

.integer = Integers are whole numbers that can have both zero, positive and negative values but no decimal values. For example, 0, -5, 10

.integer = ជា​ប្រភេទចំនួនគត់ដែរអាចផ្ទុកតម្លៃជាលេខសូន្យ,លេខវិជ្ជមាន និងលេខអវិជ្ជមាន។

W can use int for declaring an integer variable.

int id;

. Here, id is a variable of type integer.

.នេះ id ជាអថេរនៃinteger

You can declare multiple variables at once in C programming. For example

int id, age;

.The size of int is usually 4 bytes (32 bits). And, it can take 232 distinct states from -2147483648 to 2147483647.

.ទំហំនៃ int ជាធម្មតា 4 តម្លៃ (32 bit) ។ ហើយវាអាច​​ 232  ពី 2147483648 ដល់ 2147483647 ។

 

Example:

  • //integer
  • #include  < stdio.h >
  • int main()
  • {
  • int   id , age;
  • id = 1;
  • age = 19;
  • printf( "id : %d\n" ,id);
  • printf( "age : %d \n" ,age);
  • return 0;
  • }

 

Output:

  • id = 1
  • age = 19

Float and double

. float and double are used to hold real numbers.

. Float និង​ double​​​ ប្រើដើម្បីទុកលេខពិត

float salary; double price;

In C, floating-point numbers can also be represented in exponential.

នៅក្នុងភាសា C លេខអាចត្រូវបានតំណាងជាអិចស្ប៉ូណង់ស្យែលដែរ។ ឧទាហរណ៍,

flaot score = 22.442e2;

What's the difference between float and double?

-តេីភាពខុសគ្នារវាងfloatនិងdoubleខុសគ្នាយ៉ាងដូចម្តេច?

The size of float (single precision float data type) is 4 bytes. And the size of double (double precision float data type) is 8 bytes.

-ទំហំរបស់floatមានទំហំ 4bytes និងទំហំរបស់doubleមានទំហំ 8btye

 

Example:

  • //integer
  • #include  < stdio.h >
  • int main()
  • {
  • float   salary;
  • double   price;
  • printf( "salary using float : %d\n" ,salary);
  • printf( "price using double : %d \n" ,price);
  • return 0;
  • }

 

Output:

  • salary using float : -2039464564
  • price using double : 21090453409

Char

. Keyword char is used for declaring character type variables. For example.

. char ពាក្យគន្លឹះត្រូវបានប្រើសម្រាប់ប្រកាសអថេរប្រភេទតួអក្សរ។ ឧទាហរណ៍

char test = "wellcome to code Share"

. The size of the character variable is 1byte.

. ទំហំនៃអថេរcharacterគឺ 1byte ។

 

Example:

  • //integer
  • #include  < stdio.h >
  • int main()
  • {
  • char   name = 'C';
  • char   name1[] = {'C','o','d','e','s','h','a','r','e'};
  • printf( "name : %c\n" ,name);
  • printf( "name1 : %s \n" ,name1);
  • return 0;
  • }

 

Output:

  • name : h
  • name1 : Codeshare
