scanf() in C


- In C programming, scanf() is one of the commonly used function to take input from the user. The scanf() function reads formatted input from the standard input such as keyboards.

- នៅក្នុងភាសា C, scanf () គឺជាមុខងារមួយដែលត្រូវបានប្រើជាទូទៅដើម្បីទទួលការបញ្ចូលតម្លៃពីអ្នកប្រើប្រាស់។ មុខងារ scanf() អានការបញ្ចូលបានទ្រង់ទ្រាយពីការបញ្ចូលស្តង់ដាររបស់ជាក្តារចុច។

Example 1: Integer Input/Output

 

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • int testInteger;
  • printf( "Enter and integer: " );
  • scanf("%d" ,testInteger);
  • printf("Number: %d" ,testInteger);
  • return 0;
  • }

 

Output:

  • Enter an integer: 4
  • Number: 4

 

- Here, we have used %d format specifier inside the scanf() function to take int input from the user. When the user enters an integer, it is stored in the testInteger variable.

- នៅទីនេះយើងបានប្រើ %d នៅខាងក្នុង scanf() ដើម្បីទទួលការបញ្ចូលពីអ្នកប្រើប្រាស់។ នៅពេលអ្នកប្រើប្រាស់បញ្ចូលចំនួនគត់វាត្រូវបានផ្ទុកនៅក្នុងអថេtestInteger ។

Example 2 : Float and Double Input/Output

 

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • float = num1;
  • double num2;
  • printf( "Enter a number: " );
  • scanf( "%f" , &num1);
  • printf( "Enter a another number: " );
  • scanf( "%lf" , &num2);
  • printf( "Number : " , num1);
  • printf( "Another Number : " , num2);
  • return 0;
  • }

 

Output:

  • Enter a Number = 10
  • Enter a another Number = 5
  • Number = 10
  • Another Number = 5

 

- We use %f and %lf format specifier for float and double respectively

- យើងប្រើ %f និង %lf សម្រាប់ float និង double រៀងៗខ្លួន

Example 3: C Character I/O

 

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • char char;
  • printf( "Enter a Character: " );
  • scanf( "%c" , &chr);
  • printf( "Your enter a Character: %c" , chr);
  • return 0;
  • }

 

Output:

  • Enter a Character: C
  • Your enter a Character: C

 

- When a character is entered by the user in the above program, the character itself is not stored. Instead, an integer value (ASCII value) is stored.

- នៅពេលដែលតួអក្សរត្រូវបានបញ្ចូលដោយអ្នកប្រើប្រាស់នៅក្នុងកម្មវិធីខាងលើនោះតួអក្សរខ្លួនវាមិនត្រូវបានរក្សាទុកទេ។ ផ្ទុយទៅវិញតម្លៃចំនួនគត់ (តម្លៃ ASCII) ត្រូវបានរក្សាទុក។

 

- And when we display that value using %c text format, the entered character is displayed. If we use %d to display the character, it's ASCII value is printed.

-ហើយនៅពេលយើងបង្ហាញតម្លៃនោះដោយប្រើ %c តួអក្សរដែលបានបញ្ចូលត្រូវបានបង្ហាញ។ ប្រសិនបើយើងប្រើ %d ដើម្បីបង្ហាញតួអក្សរវាមានតម្លៃ ASCII ត្រូវបានបង្ហាញ។

Example 4: ASCII Value

 

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • char chr;
  • printf( "Enter a Character: " );
  • scanf( "%c" , &chr);
  • //when %c us used, a character is displayed
  • printf( "You entered %c.\n" ,chr);
  • //When %d is used, ASCII value is displayed
  • printf( "ASCII value is % d." , chr);
  • return 0;
  • }

 

Output:

  • Enter a character: C
  • You entered: C.
  • ASCII value is 103.
