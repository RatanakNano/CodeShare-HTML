Arithmetic operators (ប្រមាណវិធីនព្វន្ត)

. C supports all the basic arithmetic operators. The following table shows all the basic arithmetic operators.

. ភាសា​ C គាំទ្រគ្រប់ប្រមាវិផីមូលដ្ឋាននព្វន្ត។តារាងខាងក្រោមបង្ហាញពីប្រមាវិផីនព្វន្ធមូលដ្ឋានទាំងអស់។

Operactor Description Example
+(Adding) បូក adds two operands A+B

Relational operators(ប្រមានវិធីប្រៀបធៀប)

The following table shows all relation operators supported by C Programing.

តារាងខាងក្រោមបង្ហាញពីប្រមាវិផីប្រៀបធៀបទាំងអស់ដែរគាំទ្រដោយ C Programing

Operactor Description Example
> Check if operand on the left is greater than operand on the right A > B ( A ធំជាង B )
< Check operand on the left is smaller than right operand A < B ( A​ តូចជា​​ B )
>= check left operand is greater than or equal to right operand A >= B ( A ធំជាងឬស្មើ B )
<= Check if operand on left is smaller than or equal to right operand A <= B ( A តូចជាងឬស្មើ B )
== Check if two operand are equal A == B ( A ស្មើ B )
!= Check if two operand are not equal. A != B ( A មិនស្មើ B )

  • printf( "%d <= %d is %d" ,a , c, a <= c);
  • printf( "%d <= %d is %d" ,c , a, c <= a);
  • printf( "%d == %d is %d" ,b , a, b == a);
  • printf( "%d != %d is %d" ,c , b, c != b);

  • b == a is 1
  • c != a is 1

Logical operators(ប្រមានវិធី តក្ក)

- C language supports following 3 logical operators. Suppose a = 1 and b = 0

- ភាសា C គាំទ្រប្រមាវិផីតក្ក ៣ ខាងក្រោម។ ឧបមា a = 1 និង b = 0

Operactor Description Example
&& Logical AND (a && b) ឱ្យលទ្ធផលពិតនៅពេលលក្ខខណ្ឌទាំងពីរពិត
|| Logical OR (a || b)​​ ឱ្យលទ្ធផលពិតនៅពេលលក្ខខណ្ឌណាមួលពិត
! Logical NOT !((x>5)&&(y<5)) បើកលក្ខខណ្ឌ(x>5)&&(y<5)​នេះពិតនោះនិង​​ឱ្យលទ្ធផលទាំងពីរមិនពិត

 

Example:

  • //Logical operators(ប្រមានវិធីតក្ក)
  • #include  < stdio.h >
  • int main()
  • {
  • int   a = 5 ,b = 5 ,c = 10 ;
  • result = (a == b) && (c > b);
  • printf( "(a == b) && (c > b) is %d \n" , result);
  • result = (a == b) && (c < b);
  • printf( "(a == b) && (c < b) is %d \n" , result);
  • result = (a == b) || (c < b);
  • printf( "(a == b) || (c < b) is %d \n" , result);
  • result = (a != b) || (c < b);
  • printf( "(a != b) || (c < b) is %d \n" , result);
  • result = !(a != b);
  • printf( "!(a == b) is %d \n" , result);
  • result = !(a == b);
  • printf( "!(a == b) is %d \n" , result);
  • return 0;
  • }

 

Output:

  • (a == b) && (c > b) is 1
  • (a == b) && (c < b) is 0
  • (a == b) || (c < b) is 1
  • (a != b) || (c < b) is 0
  • !(a != b) is 1
  • !(a == b) is 0

Bitwise operators(ប្រមានវិធី​គោលពីរ)

- Bitwise operator works on bits and perform bit-by-bit operation. The truth tables for &, |, and ^ is as follows

- ប្រមាវិផី Bitwise ​ធ្វើការតិចហើយអនុវត្តប្រតិបត្តិការបន្តិចម្តងៗ។ តារាងការពិតសម្រាប់ &,|, និង^មានដូចខាងក្រោម

Operactor Description Example
& Binary AND Operator copies a bit to the result if it exists in both operands. (A & B) = 12, i.e., 0000 1100
| Binary OR Operator copies a bit if it exists in either operand. (A | B) = 61, i.e., 0011 1101
~ Binary One's Complement Operator is unary and has the effect of 'flipping' bits. (~A ) = ~(60), i.e,. -0111101
^ Binary XOR Operator copies the bit if it is set in one operand but not both. (A ^ B) = 49, i.e., 0011 0001
<< Binary Left Shift Operator. The left operands value is moved left by the number of bits specified by the right operand. A << 2 = 240, i.e., 1111 0000
>> Binary Right Shift Operator. The left operands value is moved right by the number of bits specified by the right operand. A >> 2 = 15 i.e., 0000 1111
