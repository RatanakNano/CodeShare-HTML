printf() in C


- In C Program, the printf() function is used for output. It prints the given statement to the console or output to the screen.

- នៅក្នុងភាសា​ C, printf() function ត្រូវបានប្រើសម្រាប់លទ្ធផល។ វាបោះពុម្ពសេចក្តីថ្លែងដែលបានផ្តល់ទៅកុង console រឺ​ បង្ហាញលទ្ធផលទៅអេក្រង់

- The syntax of printf() function is given below.

-មុខងារ printf () ត្រូវបានផ្តល់ជូនខាងក្រោម:

printf("format string",argument_List);

 

- The format string can be %d (integer), %c (character), %s (string), %f (float) etc.

- ខ្សែអក្សរទ្រង់ទ្រាយអាចជា% d (ចំនួនគត់),% c (តួអក្សរ),% s (ខ្សែអក្សរ),% f (ចំនូនទូសភាគ) ។ ល។

Example 1: C Output

 

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • //Display the string inside quotations
  • printf( "Welcome to Code Share" );
  • return 0;
  • }

 

Output:

  • Welcome to Code Share

Example 2: Integer Output

 

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • int testInteger = 5;
  • printf( "Number = %d" , testInteger);
  • return 0;
  • }

 

Output:

  • Number = 5

 

- We use %d format specifier to print int types. Here, the %d inside the quotations will be replaced by the value of testInteger.

- យើងប្រើ​ %d ដើម្បីបង្ហាញ int ។ នៅទីនេះ %d នៅខាងក្នុងសម្រង់នឹងត្រូវបានជំនួសដោយតម្លៃនៃ testInteger ។

Example 3: float and double Output

 

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • float number1 = 13.5;
  • double number2 = 12.4;
  • printf( "Number1 = %f" , number1);
  • printf( "Number2 = %lf" , number2);
  • return 0;
  • }

 

Output:

  • Number1 = 13.5
  • Number2 = 12.4

 

- To print float, we use %f format specifier. Similarly, we use %lf to print double values.

- ដើម្បីបង្ហាញ float,យើងប្រើ %f ។ ស្រដៀងគ្នានេះដែរយើងប្រើ %lf ដើម្បីបង្ហាញតម្លៃទ្វេ។

Example 4: Print Characters

 

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • char chr = 'a';
  • printf( "character = %c." , chr);
  • return 0;
  • }

 

Output:

  • character = a

 

To print char, we use %c format specifier.

ដើម្បីបង្ហាញ %c យើងប្រើ %c ។
