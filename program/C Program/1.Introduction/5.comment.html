Comments in C Program


- Comments in C language are used to provide information about lines of code. It is widely used for documenting code.

- Comments នៅក្នុងភាសា C ត្រូវបានប្រើដើម្បីផ្តល់ព័ត៌មាន line code ។ វាត្រូវបានគេប្រើយ៉ាងទូលំទូលាយសម្រាប់ការចងក្រងឯកសារកូដ។

- There are 2 types of comments in the C language.

- Comment ២ ប្រភេទនៅក្នុងភាសា C។

+ Single Line Comments ( comment បន្ទាត់តែមួយ )

+ Multi-Line Comments ( comment ពហុបន្ទាត់​ រឺ ពីរបន្ទាត់ឡើង )


Single Line Comments ( comment បន្ទាត់តែមួយ )

- Single line comments are represented by double slash //. Let's see an example of a single line comment in C.

- Comment បន្ទាត់តែមួយត្រូវបានតំណាងដោយសញ្ញាទ្វេ slash // ។ តោះមើលឧទាហរណ៍នៃ comment បន្ទាត់តែមួយនៅក្នុង C។

 

Example:

  • //Arithmetic operators(ប្រមាណវិធីនព្វន្ត)
  • #include  < stdio.h >
  • int main()
  • {
  • //printing information
  • printf( "Welcome to Code Share" );
  • return 0;
  • }

 

Output:

  • Welcome to Code Share

 

- Even you can place the comment after the statement. For example:

- សូម្បីតែអ្នកអាចដាក់មតិបន្ទាប់ពីសេចក្តីថ្លែងការណ៍។ ឧទាហរណ៍:

  • Welcome to Code Share

Multi-Line Comments ( comment ពហុបន្ទាត់​ រឺ ពីរបន្ទាត់ឡើង )

- Multi-Line comments are represented by slash asterisk \* ... *\. It can occupy many lines of code, but it can't be nested. Syntax:

- Comment ពហុបន្ទាត់ត្រូវបានតំណាងដោយសញ្ញាផ្កាយ \ * ... * \ ។ វាអាចកាន់កាប់កូដជាច្រើនជួរប៉ុន្តែវាមិនអាចដាក់ជាក្រុមបានទេ។ វាក្យសម្ពន្ធ៖

/*   code  to be commented  */

 

- Let's see an example of a multi-Line comment in C.

- តោះមើលឧទាហរណ៍នៃ comment ពហុបន្ទាត់នៅ C។

 

Example:

  • //Arithmetic operators(ប្រមាណវិធីនព្វន្ត)
  • #include  < stdio.h >
  • int main()
  • {
  • /* printing information
  • Multi-Line Comment */
  • printf( "Welcome to Code Share" );
  • return 0;
  • }
