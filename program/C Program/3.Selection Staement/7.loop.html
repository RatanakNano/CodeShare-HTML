C Loops


The looping can be defined as repeating the same process multiple times until a specific condition satisfies. There are three types of loops used in the C language. In this part of the tutorial, we are going to learn all the aspects of C loops.

* The looping អាចត្រូវបានកំណត់ថាធ្វើឡើងវិញនូវដំណើរការដដែលៗច្រើនដងរហូតដល់លក្ខខណ្ឌជាក់លាក់មួយ។ មានloopបីប្រភេទដែលត្រូវបានប្រើនៅក្នុងភាសា C ។ នៅក្នុងផ្នែកនៃការបង្រៀននេះយើងនឹងរៀនគ្រប់ផ្នែកនៃរង្វិលជុំ C ។

 

Why use loops in C language?

The looping simplifies the complex problems into the easy ones. It enables us to alter the flow of the program so that instead of writing the same code again and again, we can repeat the same code for a finite number of times. For example, if we need to print the first 10 natural numbers then, instead of using the printf statement 10 times, we can print inside a loop which runs up to 10 iterations.

* The looping ធ្វើឱ្យស្រូលដល់បញ្ហាដែលមានភាពស្មុគស្មាញ។ វាអាចឱ្យយើងផ្លាស់ប្តូរលំហូរនៃកម្មវិធីដូច្នេះជំនួសឱ្យការសរសេរលេខកូដដដែលម្តងហើយម្តងទៀតយើងអាចសរសេរកូដដដែលៗក្នុងចំនួនកំណត់ ។ឧទាហរណ៍ប្រសិនបើយើងត្រូវការបង្ហាញលេខធម្មជាតិចំនួន ១០ ដំបូងបន្ទាប់មកជំនួសឱ្យការប្រើ printf statement ១០ ដងយើងអាចបង្ហាញនៅខាងក្នុង loop ដែលដំណើរការរហូតដល់ ១០ ដង។

 

Benifit of loops in C

1) It provides code reusability ( វាផ្តល់នូវការប្រើប្រាស់ឡើងវិញកូដ។ ).

2) Using loops, we do not need to write the same code again and again (ដោយប្រើរង្វិលជុំយើងមិនចាំបាច់សរសេរលេខកូដដដែលៗទេ ).

3) Using loops, we can traverse over the elements of data structures (array or linked lists) {ដោយប្រើ loops យើងអាចឆ្លងកាត់ធាតុនៃ data structures (array or linked lists) ។ }.

 

C programming has three types of loops:

1.for loop

2.while loop

3.do...while loop

 

for loop in C

The for loop is used in the case where we need to execute some part of the code until the given condition is satisfied. The for loop is also called as a per-tested loop. It is better to use for loop if the number of iteration is known in advance.

* The for loop ត្រូវបានប្រើក្នុងករណីដែលយើងត្រូវការប្រតិបត្តិផ្នែកខ្លះនៃកូដរហូតដល់លក្ខខណ្ឌដែលបានផ្តល់ឱ្យត្រូវបានពេញចិត្ត។ រង្វិលជុំត្រូវបានគេហៅថា per-tested loop . វាល្អប្រសើរជាងក្នុងការប្រើរង្វិលជុំប្រសិនបើចំនួននៃ iteration ឡើងវិញត្រូវបានគេដឹងជាមុន។

The syntax of for loop in c language is given below:

  • for ( initialization; condition; incr/decr ){
  • //code to be executed
  • }

 

while loop in C

The while loop in c is to be used in the scenario where we don't know the number of iterations in advance. The block of statements is executed in the while loop until the condition specified in the while loop is satisfied. It is also called a pre-tested loop.

* ខណៈពេលដែល while loop នៅក្នុងក៏ត្រូវបានប្រើនៅក្នុង scenario ដែលយើងមិនដឹងពីចំនួននៃ​ advance ។ បណ្តុំ​ statement ត្រូវបានប្រតិបត្តិនៅក្នុង​ while loop ដែលលក្ខខណ្ឌដែលបានបញ្ជាក់នៅក្នុង loop ជុំត្រូវបានពេញចិត្ត។ វាត្រូវបានគេហៅថា pre-tested loop ។

The syntax of while loop in c language is given below:

  • While (condition;){
  • //code to be executed
  • }

 

do-while loop in C

The do-while loop continues until a given condition satisfies. It is also called post tested loop. It is used when it is necessary to execute the loop at least once (mostly menu driven programs).

* The do-while loop ធ្វើបន្តរហូតទាល់តែលក្ខខណ្ឌដែលបានផ្តល់ឱ្យពេញចិត្ត។ វាត្រូវបានគេហៅថា tested loop ។ វាត្រូវបានប្រើនៅពេលចាំបាច់ដើម្បីអនុវត្តរង្វិលជុំយ៉ាងហោចណាស់1ដង (ភាគច្រើនជាកម្ម loop menu) ។

The syntax of do-while loop in c language is given below:

  • do {
  • //code to be executed
  • }while(condition);
