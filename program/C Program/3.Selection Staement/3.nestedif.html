Nested if


- When an if statement is present inside the body of another “if” then this is called nested if.

- នៅពេលដែល​ if statement ក្នុងរ “ if ” នោះវាត្រូវបានគេហៅថា nested if។

The syntax of the if statement in C programming is

  • if ( condition1 ){
  • if ( condition2 ){
  • statements2;
  • }
  • statements1;
  • }

Example 1: Nested if

 

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • int a,b,c;
  • printf( "Enter 3 Number: " );
  • scanf( "%d\t%d\t%d" , &a,&b,&c);
  • if ( a > b ){
  • if ( a > c ){
  • printf("a is the greatest");
  • else {
  • printf("c is the greatest");
  • }
  • }
  • else {
  • if ( b > c ){
  • printf( "b is the greatest" );
  • }
  • else {
  • printf( "c is the greatest" );
  • }
  • }
  • return 0;
  • }

 

Output1:

  • Enter a Number: 1 2 3
  • c is the greatest

 

Output2:

  • Enter a Number: 3 1 2
  • a is the greatest

 

Output3:

  • Enter a Number: 1 3 2
  • b is the greatest
