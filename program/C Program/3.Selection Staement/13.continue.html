continue statement in C


The continue statement in C language is used to bring the program control to the beginning of the loop. The continue statement skips some lines of code inside the loop and continues with the next iteration. It is mainly used for a condition so that we can skip some code for a particular condition.

* continue statement នៅក្នុងភាសា C ត្រូវបានប្រើដើម្បីនាំការគ្រប់គ្រងកម្មវិធីទៅដើមនៃ loop ។  continue statement រំលងកូដមួយចំនួននៅក្នុងloopហើយបន្តជាមួយការនិយាយឡើងវិញ។ វាត្រូវបានប្រើជាចម្បងសម្រាប់លក្ខខណ្ឌមួយដូច្នេះយើងអាចរំលងកូដខ្លះសម្រាប់លក្ខខណ្ឌជាក់លាក់។

Syntax:

  • //loop statement
  • continue;
  • //some lines of the code which is to be skipped

 

The continue statement is almost always used with the if...else statement.

* សេចក្តីថ្លែងការណ៍បន្តត្រូវបានប្រើស្ទើរតែជាមួយសេចក្តីថ្លែងការណ៍ប្រសិនបើ ... ផ្សេងទៀត។

Example 1: continue statement

 

Example:

  • // Program to calculate the sum of a maximum of 10 numbers // Negative numbers are skipped from the calculation
  • // Negative numbers are skipped from the calculation
  • #include  < stdio.h >
  • int main()
  • {
  • int i;
  • double number,sum=0.0;
  • for (i = 0 ; i<= 10 ;i++){
  • printf( "Enter a n%d: " ,i);
  • scanf( "%lf" ,&number);
  • if ( number < 0.0 ){
  • continue;
  • }
  • sum += number;
  • // sum = sum + number;
  • }
  • printf( "Sum = %.2lf" ,sum);
  • return 0;
  • }

 

Output1:

  • Enter a n1: 1
  • Enter a n2: 2
  • Enter a n3: -5
  • Enter a n4: 1
  • Enter a n5: 2
  • Enter a n6: 4
  • Enter a n7: -10
  • Enter a n8: 1
  • Enter a n9: 1
  • Enter a n10: 1
  • Sum = 13.00

 

In this program, when the user enters a positive number, the sum is calculated using sum += number; statement.

* នៅក្នុងកម្មវិធីនេះនៅពេលអ្នកប្រើប្រាស់បញ្ចូលលេខវិជ្ជមានផលបូកត្រូវបានគណនាដោយ sum += number; statement។

When the user enters a negative number, the continue statement is executed and it skips the negative number from the calculation.

* នៅពេលអ្នកប្រើប្រាស់បញ្ចូលលេខអវិជ្ជមាន statement បន្តត្រូវបានប្រតិបត្តិហើយវារំលងលេខអវិជ្ជមានពីការគណនា។
