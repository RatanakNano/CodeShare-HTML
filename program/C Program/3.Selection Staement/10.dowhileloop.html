do while loop in C


The do..while loop is similar to the while loop with one important difference. The body of do...while loop is executed at least once. Only then, the test expression is evaluated.

* do..while គឺស្រដៀងនឹង while loop ដែលមានភាពខុសគ្នាសំខាន់មួយ។ តួនៃ ត្រូវបានប្រតិបត្តិយ៉ាងហោចណាស់ម្តង។ មានតែពេល test expression ត្រូវបានវាយតម្លៃ។

do while loop syntax

  • do {
  • // code to be executed
  • } while ( condition );

 

How do...while loop works?

- The body of do...while loop is executed once. Only then, the test expression is evaluated.

* do…while បានប្រតិបត្តិម្តង។ មានតែពេល test expression ត្រូវបានវាយតម្លៃ។

- If the test expression is true, the body of the loop is executed again and the test expression is evaluated.

* ប្រសិនបើ test expression នេះពិត body of the loop ត្រូវបានប្រតិបត្តិម្តងទៀតហើយ test expression ត្រូវបានវាយតម្លៃ។

- This process goes on until the test expression becomes false.

* ដំណើរការនេះបន្តរហូតដល់ test expression ក្លាយជាមិនពិត។

- If the test expression is false, the loop ends.

* ប្រសិនបើ test expression មិនពិតរង្វិលជុំត្រូវបានបញ្ចប់

Flowchart of do...while Loop

Example 1: do...while loop

 

Example:

  • // Program to add numbers until the user enters zero
  • #include  < stdio.h >
  • int main()
  • {
  • double number, sum = 0;
  • // the body of the loop is executed at least once
  • do {
  • printf( "Enter Number: " );
  • scanf( "%lf" ,&number);
  • sum+=number;
  • } while( number!= 0 );
  • printf( "Sum = %.2lf" ,sum);
  • return 0;
  • }

 

Output1:

  • Enter a number: 1
  • Enter a number: 2
  • Enter a number: - 1
  • Sum: 3

 

1) When user enter number is 1, the test expression number!= 0 is true. Hence, the body of the do while loop is executed. Sum + number and store result in sum . This prints result of sum on the screen .

* នៅពេលអ្នកប្រើប្រាស់បញ្ចូលលេខគឺ 1test expression number ! = ០ ពិត។ ដូច្នេះរាងកាយរបស់ត្ do while loop ត្រូវបានប្រតិបត្តិ។ number ! = ០ និងលទ្ធផលផ្ទុកនៅ sum។ បង្ហាញលទ្ធផលនៃការបូកនៅលើអេក្រង់។

2) Now, user enter number is 2, the test expression number != 0 is true again. The body of the do while loop is executed again. Sum + number and store result in sum again. This prints result of sum on the screen

* ឥឡូវអ្នកប្រើប្រាស់បញ្ចូលលេខគឺ 2 ដែល test expression number != 0 គឺជាការពិតម្តងទៀត។ តួនៃ do..while loop ត្រូវបានប្រតិបត្តិម្តងទៀត។ number != 0 និងលទ្ធផលផ្ទុកក្នុង sum ម្តងទៀត។ បង្ហាញលទ្ធផលនៃការបូកនៅលើអេក្រង់។

3) This process goes on until number less than 0. When number is -1, the test expression number != 0 will be false and the loop terminates.

* ដំណើរការនេះបន្តរហូតដល់លេខតូចជាង ០ ។ នៅពេលដែលលេខគឺ -1 test expression number ! = ០ នឹង មិនពិត ហើយ loop ត្រូវបានបញ្ចប់។

Example 2: do...while loop

 

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • int i = 1, number = 0;
  • printf( "Enter number: " );
  • scanf( "%d",&number );
  • do {
  • printf( "%d\n" ,(number*i));
  • i++;
  • } while( i<=number );
  • return 0;
  • }

 

Output1:

  • Enter a number: 5
  • 5
  • 10
  • 15

 

1) When I is 1 user enter number is 3, the test expression i<=number is true. Hence, the body of the do while loop is executed. Number * i and i++ . This prints result on the screen .

* នៅពេល i គឺ​ 1 អ្នកប្រើប្រាស់បញ្ចូលលេខគឺ 3 test expression i<=number ពិត។ ដូច្នេះរាងកាយរបស់ត្ do while loop ត្រូវបានប្រតិបត្តិ។ number * i និង i++ ។ បង្ហាញលទ្ធផលនៅលើអេក្រង់។

2) Now i is 2, the test expression i<=number is true again. The body of the do while loop is executed again. Number*i and i++. This prints result on the screen .

* ឥឡូវ i គឺ​ 2, test expression i<=number គឺជាការពិតម្តងទៀត។ តួនៃ do..while loop ត្រូវបានប្រតិបត្តិម្តងទៀត។ number*i និង i++។ បង្ហាញលទ្ធផលនៃការបូកនៅលើអេក្រង់។

3) This process goes on until i less than number and Equal number. When i is 3, the test expression i<=number will be true ,then Number*i and i++. This prints result on the screen and the loop terminates.

* ដំណើរការនេះបន្តរហូតដល់i តូចជាង number ឬ ស្មេីរ number ។ នៅពេលដែល i គឺ 3 test expression i<=number នឹងពិតហើយ loop number*i និង i++។ បង្ហាញលទ្ធផលនៃការបូកនៅលើអេក្រង់ និង​ loopត្រូវបានបញ្ចប់។
