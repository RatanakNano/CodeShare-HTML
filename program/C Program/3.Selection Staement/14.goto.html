goto statement in C


The goto statement is known as jump statement in C. As the name suggests, goto is used to transfer the program control to a predefined label. The goto statment can be used to repeat some part of the code for a particular condition. It can also be used to break the multiple loops which can't be done by using a single break statement. However, using goto is avoided these days since it makes the program less readable and complecated.

* Goto statement ត្រូវបានគេស្គាល់ថាជា statement នៅក្នុង C។ ដូចត្រូវបានប្រើដើម្បីធ្វើម្តងទៀតផ្នែកខ្លះនៃកូដសម្រាប់លក្ខខណ្ឌជាក់លាក់មួយ។ វាក៏អាចត្រូវបានប្រើដើម្បីបំបែក​ loop ជាច្រើនដែលមិនអាចធ្វើបានដោយប្រើ statement តែមួយ។ ទោះជាយ៉ាងណាក៏ដោយការប្រើប្រាស់ goto ត្រូវបានជៀសវាងនៅពេលចាប់តាំងពីវាធ្វើឱ្យកម្មវិធីមិនសូវអាចអាននិងចងក្រងបាន។

Syntax:

  • label:
  • //some part of the code;
  • goto label;

Example 2

The only condition in which using goto is preferable is when we need to break the multiple loops using a single statement at the same time. Consider the following example.

* លក្ខខណ្ឌតែមួយគត់ដែលការប្រើប្រាស់ goto គឺល្អប្រសើរជាងមុនគឺនៅពេលដែលយើងត្រូវការបំបែក loop ច្រើនដោយប្រើ statement តែមួយក្នុងពេលតែមួយ។ ពិចារណាឧទាហរណ៍ខាងក្រោម។

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • int i,j,k;
  • for ( i= 0;i< 10; i++ ){
  • for ( j= 0;j< 5; j++ ){
  • for ( k= 0;k< 3; k++ ){
  • printf( "%d %d %d\n" ,i,j,k);
  • if ( j==3 ){
  • goto out;
  • }
  • }
  • }
  • }
  • out:
  • printf( "came out of the loop" );
  • return 0;
  • }
