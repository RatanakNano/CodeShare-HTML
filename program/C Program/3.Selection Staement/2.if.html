If Statement


-If statements in C. The ability to control the flow of your program, letting it make decisions on what code to execute, is valuable to the programmer. The if statement allows you to control if a program enters a section of code or not based on whether a given condition is true or false.

- If នៅក្នុងភាសា C ។ សមត្ថភាពក្នុងការគ្រប់គ្រងនៃកម្មវិធីរបស់អ្នកដោយអនុញ្ញាតឱ្យវាធ្វើការសម្រេចចិត្តលើកូដដែលត្រូវប្រតិបត្តិគឺមានតម្លៃសម្រាប់អ្នកសរសេរកម្មវិធី។ if statement ឱ្យអ្នកត្រួតពិនិត្យប្រសិនបើកម្មវិធីបញ្ចូលផ្នែកនៃកូដឬមិនផ្អែកលើលក្ខខណ្ឌដែលបានផ្តល់គឺពិតឬមិនពិត។

 

The syntax of the if statement in C programming is

  • if ( condition ){
  • //statements to be executed if the test expression is true
  • statements;
  • }

 

- The if statement evaluates the test expression inside the parenthesis ().

- If statement វាយតម្លៃកន្សោមតេស្តនៅខាងក្នុងវង់ក្រចក () ។

 

+ If the test expression is evaluated to true, statements inside the body of if are executed.

+ ​ប្រសិនបើកន្សោមតេស្តត្រូវបានវាយតម្លៃទៅជាការពិតសេចក្តីថ្លែងការណ៍នៅខាងក្នុងខ្លួនប្រសិនបើត្រូវបានប្រតិបត្តិ។

+ If the test expression is evaluated to false, statements inside the body of if are not executed.

+ if សាកល្បង តេស្តវាយតម្លៃមិនពិត statements នៅខាងក្នុង body របស់ if មិនត្រូវបានអនុវត្តទេ។

Expression is True Expression is Fail
  • int a = 1;
  • if ( a < 5 ) {
  • // codes (work)
  • }
  • //code after if
  • int a = 1;
  • if ( a < 5 ) {
  • // codes
  • }
  • //code after if (work)

Example:

  • #include  < stdio.h >
  • int main()
  • {
  • int m, n;
  • printf( "Enter M: " );
  • scanf("%d" ,&m);
  • printf( "Enter N: " );
  • scanf("%d" ,&n);
  • if ( m == n ) {
  • printf( "m and n are equal." );
  • }
  • printf( "m and n are not equal" );
  • return 0;
  • }

 

Output1:

  • // m =10 and n = 10
  • m and n are equal.

 

- When we input m = 40 and n = 40 if expression is evaluated to true because m = n

- នៅពេលយើងបញ្ចូល m = 40 និង n = 40 ប្រសិនបើការបញ្ចេញមតិត្រូវកាវាយតម្លៃទៅជាការពិតពីព្រោះ m = n

 

Output2:

  • // m =10 and n = 5
  • m and n are not equal.

 

- When we input m = 40 and n = 30 if expression is evaluated to fail because m != n

- នៅពេលយើងបញ្ចូល m = 40 និង n = 30 ប្រសិនបើការបញ្ចេញមតិមិនត្រូវកាវាយតម្លៃទៅជាការមិនពិតពីព្រោះ m != n ។
